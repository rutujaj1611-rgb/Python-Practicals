# 🐍 Python Programming Lab Practicals

This repository includes a set of Python lab experiments created as part of an academic curriculum. It focuses on building a strong foundation in Python by covering essential programming concepts such as data types, decision-making, functions, object-oriented programming, file operations, and error handling.

---

## 📌 Experiments Overview

### 🔹 Experiment 1: Data Types and Operators in Python

Study of various Python data types and implementation of different operators like arithmetic, relational, and logical operators.

---

### 🔹 Experiment 2: Conditional Statements

Implementation of decision-making statements including `if`, `if-else`, `nested if`, and `if-elif-else`.

---

### 🔹 Experiment 3: String Manipulation

Working with strings including slicing, searching, and usage of built-in string functions.

---

### 🔹 Experiment 4: Lists

Understanding list operations such as adding, removing, sorting, and iterating elements.

---

### 🔹 Experiment 5: Tuples

Exploring tuple properties, indexing, slicing, and operations like count and concatenation.

---

### 🔹 Experiment 6: Dictionaries

Implementation of dictionary operations including insertion, modification, deletion, and traversal of elements.

---

### 🔹 Experiment 7: Functions

Usage of built-in functions and creation of user-defined functions to perform specific tasks.

---

### 🔹 Experiment 8: Classes and Objects

Introduction to object-oriented programming concepts with implementation of classes and objects.

---

### 🔹 Experiment 9: Inheritance (Single & Multiple)

Understanding inheritance types in Python with practical examples of single and multiple inheritance.

---

### 🔹 Experiment 10: File Handling

Performing file operations such as reading, writing, appending, and managing files.

---

### 🔹 Experiment 11: Exception Handling

Handling runtime errors using `try`, `except`, `finally`, and creating custom exceptions.

---
